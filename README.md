Currency Converter 💱

A simple and intuitive web application that allows users to convert amounts between different currencies in real time.
The app features a clean, responsive interface where users can:

- Enter an amount in their chosen source currency.
- Select both source and target currencies.
- Instantly view the converted value using up‑to‑date exchange rates.

  Features

- 🌍 Supports multiple global currencies.
- ⚡ Real‑time conversion with accurate rates.
- 📱 Responsive design for desktop, tablet, and mobile.
- 🎨 Clean UI for easy navigation and quick results.
  Tech Stack
- Frontend: HTML, CSS, JavaScript
- API Integration: Exchange rate API for live data
- Design: Responsive layout with modern UI styling
